# JNote
Java Windows Notepad Clone

Clon del bloc de notas de Windows en Java (Swing). Las siguientes características faltan por ser implementadas:
  - Imprimir
  - Atajos de teclado
  - Edición -> Ir a...
  - Edición -> Buscar siguiente
  - Formato -> Ajuste de línea
  - Formato -> Fuente
  - Ver -> Barra de estado
